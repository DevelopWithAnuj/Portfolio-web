# Personal Portfolio Website V3

This repository contains the source code for my personal portfolio, a modern and responsive website designed to showcase my skills, projects, and journey as a frontend developer. The project emphasizes clean design, performant UI, and a developer-centric approach.

## Tech Stack

This project is built with a focus on web fundamentals, enhanced with modern web platform features.

-   **HTML5:** Semantic and accessible markup.
-   **CSS3:**
    -   **Responsive Design:** Using `clamp()` and media queries for a seamless experience on all devices.
    -   **Glassmorphism:** Modern UI trend for a refined, premium feel.
    -   **Custom Properties:** For maintainable and scalable styling.
    -   **Animations:** Subtle animations and transitions to improve user experience.
-   **JavaScript (ES6+):**
    -   **DOM Manipulation:** For interactive elements and dynamic content.
    -   **Page Loader & Redirects:** Smooth page transitions.
-   **No Frameworks:** A conscious decision to showcase a strong understanding of core web technologies.

---

## Features

-   **Modern & Premium UI:** A refined design with a focus on typography, color contrast, and glassmorphism.
-   **Fully Responsive:** Optimized for mobile, tablet, and desktop viewing.
-   **Developer-Focused Content:** The landing page is written to appeal to a technical audience.
-   **Improved UX:** Enhanced button hover and focus states for better interactivity.
-   **Performant:** Lightweight and fast-loading, with no heavy dependencies.
-   **Project Showcase:** A dedicated section to display my work.
-   **Contact & Social Links:** Easy ways to connect.

## Getting Started

To run this project locally, you can simply open the `index.html` file in your web browser.

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/your-username/your-repo-name.git
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd your-repo-name
    ```
3.  **Open `index.html` in your browser.**

No installation or build steps are required.
